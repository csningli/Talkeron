Introduction

Comment: Introduce the keys of developing and using of Talkeron.

Talkeron is a console-gui-mixed application for task management. In console, user can type in the name of talkeron assists and running configured value of the parameters. By clicking "Enter" key, the typed command will be compiled and start the task block of the corresponding assist. In the task block, the default value of parameters are extracted from the file which has the assist description, and it is replaced by the running configured value.

When an assist has been compiled correctly, a corresponding task block is created in the task area. By clicking the buttons in the task blocks, user can start the corresponding action. There are two kinds of actions: the binded actions "run", "redo" and "undo" of the assist; the hand actions which link to other assists.

TalkeronUsage

Comment: Introduce the usage of Talkeron.

A typical use of Talkeron contains the following steps:

  1. start Talkeron, if it is not running yet;
  2. type in the command and press "Enter";
  3. fill in or change the values of the parameters;
  4. click an action button.

TalkeronUsage | ConsoleUsage

Comment: Introduce the commands acceptable by Talkeron console.

A command acceptable by Talkeron console has two parts:

  1. Assist indicator. A dot separated string which indicates an assist.
  2. Running configured values of parameters. A list of terms of format "[parameter name]=[running configured value]". This part is optional.

the "Assist" directory under TalkeronHome.

For example, there is a built-in auxiliary assist "Talk" which is described in file "TalkeronHome/Assist/Talk/Talk.talk". To refer to assist "Talk", user can type "Talk" directly in the console. Furthermore, there is a child assist "Alist" in "Talk". Assist "Alist" is described in file "TalkeronHome/Assist/Talk/Alist/Alist.talk", and is referred by indicator "Talk.Alist".

TalkeronUsage | TaskBlockUsage

Comment: Introduce the allowed operations of task blocks.

In a created task block, there are two things a user can do:

  1. change the parameter values;
  2. and click the action buttons.

TalkeronDevelopment

Comment: Introduce the important ideas in Talkeron applications.

About the development of Talkeron, the two most important things are "compiling of the typed command" and "generating of the task block".
